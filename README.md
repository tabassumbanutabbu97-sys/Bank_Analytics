# Bank_Analytics
Project Overview:
This project focuses on analyzing bank loan data using Excel, SQL, Power BI, and Tableau to derive meaningful business insights. The objective is to understand customer behavior, loan performance, and risk factors.

Project Objectives:
- To analyze bank loan data and identify key factors influencing loan approval and rejection
- To understand customer demographics and their impact on loan distribution
- To evaluate loan performance and detect potential risk or default patterns
- To analyze transaction trends and customer behavior for better decision-making
- To clean, transform, and manage financial data efficiently using Excel and SQL
- To build interactive dashboards using Power BI and Tableau for clear data visualization
- To generate actionable insights that can help banks improve loan strategies and reduce risk


KPI Used:
- Year wise loan amount Stats
- Grade and sub grade wise revol_bal
- Total Payment for Verified Status Vs Total Payment for Non Verified Status
- State wise and month wise loan status
- Home ownership Vs last payment date stats

Dashboard Features:
- KPI Cards
- Payment Recency Analysis
- Home Ownership Risk Comparison
- Interactive Filters

Tools Used:
- Excel (Power Query, Pivot Tables)
- Power BI (DAX, Dashboard)
- Tableau (Calculated Fields, Dashboard)
- Mysql(KPI's Queries)

---

Key Insights:
- Homeowners show better payment discipline
- Renters have higher payment delays
- Payment recency is a strong risk indicator
